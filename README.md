# Hybrid-keepAccounts

## 基本项目目录
```
|__ Item
	|__ dist
		|__ bundle.js		// webpack打包完成后的js
		|__ index.html		// 主页面
		|__ lib				// 复制过来的库
	|__ src					// 放webpack打包的index.js
		|__ index.js		// 主要逻辑
	|__ node_modules		// npm下载的包
	|__ webpack.config.js	// webpack的配置文件
	|__ .gitignore			// 项目忽略文件，上传到github or gitee时不上传的目录
	|__ README.md		// 项目说明文件
	|__ LICENSE			// 开源协议
```
